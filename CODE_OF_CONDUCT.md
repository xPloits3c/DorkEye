# Code of Conduct

## Our Pledge
We are committed to providing a respectful, inclusive, and harassment-free experience for everyone participating in this project.

## Expected Behavior
- Be respectful and professional
- Accept constructive criticism
- Use welcoming and inclusive language
- Respect differing viewpoints

## Unacceptable Behavior
- Harassment, discrimination, or abuse
- Trolling or personal attacks
- Publishing private information without consent
- Any illegal or unethical activity

## Enforcement
Project maintainers have the right to remove content or contributors that violate this code.

## Reporting Issues
If you experience or witness unacceptable behavior, please report it privately to the project maintainer.

---

This Code of Conduct applies within all project spaces.
